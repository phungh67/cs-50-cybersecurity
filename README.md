## 📖 About This Repository

The goal of this repository is to serve as a centralized, easily searchable "second brain" for complex IT, Systems engineering, and Cybersecurity concepts. 

Currently, the documentation spans from low-level Operating System concepts and boot sector layouts (MBR/UEFI) to high-level Distributed Systems (including MapReduce architecture in Golang) and Data Privacy paradigms.

## 🗂️ Repository Structure

The documentation is logically divided into several core domains. Click any folder to dive into the specific module:

* **[📁 Computer Network](./Computer%20Network/)** - Notes on networking protocols, architecture, and network-level defense.
* **[📁 Computer security](./Computer%20security/)** - [WIP Masterpiece] Core security concepts, system vulnerabilities, CI/CD pipeline security, and mitigation strategies.
* **[📁 Cybersecurity-CS50](./Cybersecurity-CS50/)** - [On Hiatus] Foundational notes and takeaways from the CS50 Cybersecurity curriculum.
* **[📁 Data-privacy](./Data-privacy/)** - [Complete] Concepts related to data protection, including k-anonymity, linkage attacks, quasi-identifiers, and differential privacy.
* **[📁 Distributed-systems](./Distributed-systems/)** - Notes on distributed computing paradigms, including MapReduce architecture.
* **[📁 Operating-systems](./Operating-systems/)** - Deep dives into OS functionality, disk space utilization, file systems, and boot processes (MBR & UEFI).
* **[📁 Real-tume-systems](./Real-tume-systems/)** - Scheduling algorithms (like Rate Monotonic Scheduling), timing constraints, and RTOS exercises.

### 📄 Notable Standalone Files
* [`Attack-SQL-inject.md`](./Attack-SQL-inject.md) - Detailed breakdown of SQL Injection attacks and prevention.
* [`Barrier in high-income countries.md`](./Barrier%20in%20high-income%20countries.md) - Contextual/policy research notes.

## 🚀 Getting Started

Since this repository is entirely composed of Markdown (`.md`) files and supplementary images, you do not need to install any specific software dependencies to read it. 

1. Simply clone the repository:
   ```bash
   git clone [https://github.com/phungh67/cs-50-cybersecurity.git](https://github.com/phungh67/cs-50-cybersecurity.git)
   ```
2. Open the folder in your favorite Markdown editor to easily navigate the linked documents and view the diagrams.
