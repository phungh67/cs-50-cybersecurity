Welcome to my personal knowledge base and study repository. This repository contains a curated collection of Markdown documents, notes, and architectural diagrams gathered throughout my Master's studies in **Computer Systems and Cybersecurity**, as well as supplemental learning from **Chalmers University of Technology**, **Coursera** (including CS50), and various other sources.

## 📖 About This Repository

The goal of this repository is to serve as a centralized, easily searchable "second brain" for complex IT, Systems engineering, and Cybersecurity concepts. 

Currently, the documentation spans from low-level Operating System concepts and boot sector layouts (MBR/UEFI) to high-level Distributed Systems (MapReduce architecture) and Data Privacy paradigms.

## 🗂️ Repository Structure

The documentation is logically divided into several core domains:

* **`📁 Computer Network/`** - Notes on networking protocols, architecture, and network-level defense.
* **`📁 Computer security/`** - Core security concepts, system vulnerabilities, and mitigation strategies.
* **`📁 Cybersecurity-CS50/`** - Foundational notes and takeaways from the CS50 Cybersecurity curriculum.
* **`📁 Data-privacy/`** - Concepts related to data protection, including k-anonymity, linkage attacks, quasi-identifiers, and re-identification threats.
* **`📁 Distributed-systems/`** - Notes on distributed computing paradigms, including MapReduce architecture.
* **`📁 Operating-systems/`** - Deep dives into OS functionality, disk space utilization, file systems, and boot processes (MBR & UEFI).
* **`📁 Real-tume-systems/`** *(Note: covers Real-Time Systems)* - Scheduling algorithms, timing constraints, and RTOS concepts.

### 📄 Notable Standalone Files
* `Attack-SQL-inject.md` - Detailed breakdown of SQL Injection attacks and prevention.
* `Barrier in high-income countries.md` - Contextual/policy research notes.
* **Diagrams & Architecture:** Various `.png` files mapping out MBR/UEFI layouts, Linkage attacks, MapReduce structures, and more to supplement the markdown notes.
